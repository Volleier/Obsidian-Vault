
统计学为人工智能提供了从数据中估计参数、评估模型表现和做出可量化推断的系统方法。如果说概率论从已知分布"正向"推导观测结果的分布，统计学则从观测数据"反向"推断数据生成过程——这正是监督学习的基本逻辑。从最大似然估计到交叉验证，统计学构成了机器学习模型训练和评估的理论支柱。

# 描述性统计

描述性统计是数据分析的第一步——在训练模型之前，理解数据的"形状"至关重要。均值 $\bar{x} = \frac{1}{n}\sum_i x_i$ 给出数据的"中心位置"，但它对异常值极为敏感。中位数是排序后居中位置的值，不受极端值影响——在收入数据分析等场景中，中位数往往比均值更有参考价值。众数是出现频率最高的值，在分类特征的探索性分析中直观有用。

方差 $s^2 = \frac{1}{n-1}\sum (x_i - \bar{x})^2$ 和标准差 $s = \sqrt{s^2}$ 衡量数据的离散程度。标准差与原始数据同量纲，比方差更易解释——"数据的典型偏离量约为 $s$"比"方差为 $s^2$"直观得多。四分位距（IQR = Q3 - Q1）覆盖中间 50% 的数据，不受极端值影响，是箱线图的绘制基础。

相关性分析是特征工程中的重要工具。皮尔逊相关系数 $r = \frac{\sum(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum(x_i - \bar{x})^2 \sum(y_i - \bar{y})^2}}$ 衡量线性关联的强度，取值 $[-1,1]$。若 $|r| \approx 1$，两个特征高度线性相关——其中一个可能是冗余的。但 $r$ 不捕获非线性关系；斯皮尔曼等级相关系数基于排序而非原始值，能检测单调非线性关联。

# 最大似然估计（MLE）

最大似然估计是机器学习中最核心的参数估计方法。它的核心直觉极其朴素：选择那些使"我们实际观测到的数据"出现概率最大的参数值作为估计。

给定独立同分布样本 $\mathcal{D} = \{x_1, \dots, x_n\}$，似然函数为 $L(\theta) = p(\mathcal{D}|\theta) = \prod_{i=1}^n p(x_i|\theta)$——每个样本出现的概率的乘积。取对数将连乘转化为连加，得到对数似然 $\ell(\theta) = \log L(\theta) = \sum_{i=1}^n \log p(x_i|\theta)$，优化这个函数等价于优化原始似然但对数形式数值更稳定。

最大似然估计量 $\hat{\theta}_{\text{MLE}} = \arg\max_\theta \ell(\theta)$ 通过令对数似然的导数为零求解。举两个典型例子：高斯均值的 MLE 就是样本均值 $\hat{\mu} = \frac{1}{n}\sum x_i$，自然且直观；高斯方差（有偏）的 MLE 为 $\hat{\sigma}^2 = \frac{1}{n}\sum (x_i - \hat{\mu})^2$——注意分母为 $n$ 而非无偏估计的 $n-1$，这体现了 MLE 可能带有轻微偏差。

MLE 具有优秀的理论性质：在大样本下，MLE 是渐近无偏的（一致性）、方差达到 Cramér-Rao 下界（渐近有效性）、服从渐近正态分布。这些性质意味着——给定足够数据——MLE 几乎是最优的估计方法。分类器的交叉熵损失和回归的均方误差，本质上都是对应分布假设下的负对数似然。

# 最大后验估计（MAP）

当数据量不足或模型复杂度过高时，仅靠数据（MLE）可能导致过拟合——模型学会了数据中的噪声而非常规规律。最大后验估计（MAP）通过引入关于参数的先验信念来缓解这一问题，在似然之外加上一道"信心约束"。

目标函数为：$\hat{\theta}_{\text{MAP}} = \arg\max_\theta \log p(\mathcal{D}|\theta) + \log p(\theta)$。对比 MLE 的目标——只有 $\log p(\mathcal{D}|\theta)$——MAP 多出的 $\log p(\theta)$ 正是正则化的概率解释。选取高斯先验 $p(\theta) \propto e^{-\lambda\|\theta\|^2}$，则 $\log p(\theta) = -\lambda\|\theta\|^2$ 恰好对应 L2 正则化；选取拉普拉斯先验 $p(\theta) \propto e^{-\lambda\|\theta\|_1}$，则 $\log p(\theta) = -\lambda\|\theta\|_1$ 恰好对应 L1 正则化。这意味着 Ridge 回归和 Lasso 回归本质上就是在做 MAP 估计——只是先验分布的选择不同。

当样本量增大时，$\log p(\mathcal{D}|\theta)$ 的量级随 $n$ 增长（$\sim O(n)$），而先验 $p(\theta)$ 的量级不随 $n$ 变化。因此在大数据下，MAP 渐近收敛于 MLE——数据的规模最终主导了推断。

# 偏差-方差权衡

偏差-方差权衡解释了机器学习中最核心的现象——过拟合和欠拟合——的数学根源。对于平方损失，任何模型的泛化误差都可以分解为三个不可约部分：

$$\mathbb{E}[(y - \hat{f}(x))^2] = (\text{Bias}[\hat{f}(x)])^2 + \text{Var}[\hat{f}(x)] + \sigma^2$$

偏差是模型在所有可能的训练集上的平均预测与真实值之间的系统性偏离。它的直观表现：高偏差的模型"太简单"，连训练数据中的基本规律都捕捉不到——欠拟合。线性模型在非线性数据上的表现就是经典的高偏差场景。

方差是模型对训练数据变化的敏感度——用不同的训练集会得到截然不同的预测。高方差的模型过度记忆了训练集中的细节（包括噪声），换一组数据表现就大幅波动——过拟合。深度决策树和高阶多项式回归是高方差的典型。

固有噪声 $\sigma^2$ 是数据生成过程中不可消除的随机性，即使无限数据也无法消除。这三者之间存在天然的权衡——增加模型复杂度（如增加网络层数）会降低偏差（模型能捕获更复杂的模式）但增加方差（模型更容易被噪声扰动）。正则化通过限制模型复杂度，用适度提升偏差的代价来大幅降低方差，在两者之间寻求总误差的最小值。

# 交叉验证

交叉验证是评估模型泛化能力的标准工具。它将数据分成训练和验证两个互不重叠的部分，用训练部分拟合模型、用验证部分估计泛化误差。这样做的核心原因在于——如果直接在训练集上评估模型，会严重低估真实误差（因为模型已经"见过"这些数据了）。

K 折交叉验证将数据均匀分成 $K$ 份：每次用其中 $K-1$ 份训练模型，剩下的一份作为验证集。这个过程重复 $K$ 遍——每份数据都有机会当一次验证集。最终取 $K$ 次验证性能的均值作为泛化误差估计。$K=5$ 或 $K=10$ 在实践中最常用，兼顾了评估稳定性和计算开销。

留一交叉验证（LOOCV）是 $K=n$ 的极端——每次只留一个样本验证，训练 $n$ 次模型。它的偏差最小（几乎每次都用到了全部样本训练），但方差较大（$n$ 个几乎相同的模型之间的性能波动）且计算代价过高。

分层 K 折交叉验证在划分时保持每个折中各类别样本的比例与总体一致。这对于类别不平衡的数据特别重要——如果不加分层，某个折可能完全缺失少数类样本，评估结果会有严重的偏差。

# 假设检验与置信区间

假设检验为机器学习中的模型比较和实验评估提供了统计上严格的决策框架。在 A/B 测试中，我们想知道"模型 A 的效果优于模型 B"这个结论有多少把握——假设检验用 p 值量化了"如果效果相同（零假设 $H_0$），观测到当前差异或更大差异的概率"。

p 值小于预设的显著水平 $\alpha$（常用 0.05）时，我们拒绝 $H_0$，认为差异是统计显著的。但 p 值本身不衡量效果大小——在超大样本下，微小的、无实际意义的差异也可能达到统计显著。因此 p 值需要结合效应量共同解释。

第一类错误（假阳性）是拒绝了一个真的 $H_0$，概率恰好等于 $\alpha$。第二类错误（假阴性）是未能拒绝一个假的 $H_0$——模型确实有改进但我们检测不到。统计功效 $1-\beta$ 衡量正确检出差别的能力，它与样本量和效应量正相关——收集更多数据是最直接的提高功效的方法。

置信区间则提供了参数估计的不确定性范围。在大数定律和中心极限定理的保证下，样本均值的 95% 置信区间近似为 $\bar{x} \pm 1.96 \cdot \frac{s}{\sqrt{n}}$。区间的宽度随样本量增大而缩小（$\sim 1/\sqrt{n}$）——将精度提高一倍需要四倍的数据。

# 模型评估指标

选择合适的评估指标取决于具体任务的需求和代价。分类任务中，最直观的准确率（Accuracy）在所有类别样本量一致时是合理的，但面对类别不平衡时会给出误导性的乐观结果——如果 99% 的样本是正类，全判正类就能达到 99% 准确率。

精确率（Precision = $\frac{TP}{TP + FP}$）关注"判为正类的样本中有多少是真正正类"——当假阳性的代价很高时（如垃圾邮件误判为正常邮件），精确率是关键指标。召回率（Recall = $\frac{TP}{TP + FN}$）关注"所有真实正类中有多少被成功找到"——当漏检的代价很高时（如疾病筛查），召回率是关键指标。

F1 分数是精确率和召回率的调和平均：$F1 = 2 \cdot \frac{P \cdot R}{P + R}$。调和平均而非算术平均的原因在于，它惩罚极端不平衡——精确率很高但召回率很低时，F1 会很低，这促使模型在两个指标上都保持合理水平。

ROC 曲线以假阳率（FPR）为横轴、真阳率（TPR 即召回率）为纵轴，展示了分类器在不同决策阈值下的性能全貌。AUC（曲线下面积）是对分类器整体判别能力的综合度量——AUC = 1 表示完美的判别能力，AUC = 0.5 表示等价于随机猜测。AUC 对类别不平衡不敏感，因此在评估排序能力时有独特优势。

回归任务则使用均方误差 MSE = $\frac{1}{n}\sum (y_i - \hat{y}_i)^2$ 和平均绝对误差 MAE = $\frac{1}{n}\sum |y_i - \hat{y}_i|$。MSE 对大误差的惩罚显著强于 MAE——一个远离预测值的离群点对 MSE 的影响远比几个中等误差大。$R^2 = 1 - \frac{\sum (y_i - \hat{y}_i)^2}{\sum (y_i - \bar{y})^2}$ 则表示模型解释了多少比例的目标变量方差，$R^2 = 0.8$ 意味着模型捕捉了 80% 的变化规律。

# 大数定律与中心极限定理

大数定律为"更多数据 = 更好估计"提供了数学保证：当样本量趋于无穷时，样本均值以概率 1（强定律）或依概率（弱定律）收敛到总体均值。这解释了为什么随着训练数据增加，模型参数估计的方差会减小——越多数据，均值估计越稳定。

中心极限定理（CLT）则是另一个基础性结论：独立同分布随机变量的样本均值 $\bar{X}_n$ 经标准化后，无论原始分布的形状如何，都趋向于标准正态分布：
$$\frac{\bar{X}_n - \mu}{\sigma / \sqrt{n}} \xrightarrow{d} \mathcal{N}(0, 1)$$

CLT 的威力在于它的普适性——即使数据的原始分布严重偏离正态（甚至呈指数衰减或完全有界），均值的抽样分布也会在足够样本后趋向正态。这个结论支撑了几乎所有常用统计推断方法——置信区间、假设检验、模型参数的标准误估计——因为只要样本量足够大，正态近似就是有效的。它也从一个侧面解释了为什么 Bagging 集成方法（通过对多个子模型的预测取平均）能有效降低方差：每次重采样的预测是独立的随机变量，平均后的方差按 $1/B$ 缩小（$B$ 为集成个数），分布趋向更集中的正态。
