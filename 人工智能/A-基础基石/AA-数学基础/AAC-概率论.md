
概率论为人工智能提供了处理不确定性的数学语言。现实世界的数据从来不是精确的——传感器有噪声、标签有人为错误、未来事件本身就不确定。概率论将"不确定性"量化为可计算的数值，使机器学习模型不仅能做出预测，还能告诉我们对预测的把握有多大。从贝叶斯分类器到生成式对抗网络，概率的思维贯穿 AI 的每一个子领域。

# 概率空间与公理

概率空间是建立所有概率推理的基础框架，由三个要素构成：样本空间 $\Omega$（所有可能结果的集合）、事件集合 $\mathcal{F}$（我们关心的子集），以及概率测度 $P$。柯尔莫哥洛夫于 1933 年提出的三条公理为概率赋予了严格的数学定义——对任意事件 $A$，有 $0 \leq P(A) \leq 1$；$P(\Omega) = 1$（总概率为1）；若事件互不相容，则 $P(\bigcup_i A_i) = \sum_i P(A_i)$。

条件概率是在已有信息下重新评估不确定性的工具，其定义为 $P(A|B) = \frac{P(A \cap B)}{P(B)}$（$P(B) > 0$）。在机器学习中，条件概率无处不在：$P(y|\mathbf{x})$ 表示给定输入特征时输出标签的概率分布——这正是分类器输出的东西。全概率公式 $P(A) = \sum_i P(A|B_i)P(B_i)$ 则将复杂的边际概率分解为简单条件概率的加权和。

# 贝叶斯定理

贝叶斯定理是机器学习的理论基石之一。它将条件概率颠倒过来：
$$P(A|B) = \frac{P(B|A) P(A)}{P(B)}$$

在 AI 的语境下，这个简单的公式被重新诠释为"从数据中学习"的数学框架：
$$P(\theta|\mathcal{D}) = \frac{P(\mathcal{D}|\theta) P(\theta)}{P(\mathcal{D})}$$

这里的每个部分都有直观的含义：$P(\theta)$ 是先验——在看到数据之前对模型的已有信念，例如认为权重应该比较小（高斯先验）；$P(\mathcal{D}|\theta)$ 是似然——给定当前模型后数据出现的可能性，这通常是损失函数的概率解释；$P(\theta|\mathcal{D})$ 是后验——在看到数据后对模型的更新信念，综合了先验知识和数据证据。$P(\mathcal{D})$ 是边际似然，充当归一化常数，同时也是模型比较中贝叶斯因子的关键组成部分。

贝叶斯定理体现了一个优雅的学习逻辑：初始信念（先验）+ 新证据（数据）= 更新信念（后验）。这个过程可以迭代——后验成为下一轮学习的先验，如此逐步逼近真实分布。当样本量趋于无穷时，后验集中在真实参数周围，数据的力量最终压倒了先验。

# 随机变量与常见离散分布

随机变量将不确定的"事件"转化为确定的"数值"，使我们能够进行数学计算。离散随机变量的取值是可数的，由概率质量函数（PMF）$P(X = x)$ 描述，所有取值的概率之和为 1。连续随机变量的取值是连续的，由概率密度函数（PDF）$p(x)$ 描述，积分为 1，但任一点的概率为零——概率只能通过区间积分给出。

伯努利分布是描述一次二元试验的最简分布：$P(X=1) = p$，$P(X=0) = 1-p$。在机器学习的二分类问题中，模型的输出本质上就是对一个伯努利分布的参数 $p$ 的估计。

二项分布 Binomial$(n,p)$ 将 $n$ 次独立伯努利试验中成功次数建模为 $P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$。这个分布在 A/B 测试中用于判断两组转化率的差异是否有统计意义。

多项分布和分类分布将伯努利的二元结果推广到多类别场景。分类分布在自然语言处理中无处不在——语言模型预测下一个词时，输出层 softmax 给出的就是类别概率向量，即 Categorical$(\mathbf{p})$ 的参数 $\mathbf{p}$。

泊松分布 $P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$ 描述"单位时间内随机事件发生的次数"——在推荐系统中用于建模用户点击次数的分布。

# 常见连续分布

连续分布在建模真实世界的测量值、噪声和潜在变量时不可或缺。与离散分布不同，连续分布中任一点的"概率"是无穷小的——实际有意义的是落在某区间内的概率。

高斯分布（正态分布）$\mathcal{N}(\mu, \sigma^2)$ 的密度函数 $p(x) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp(-\frac{(x-\mu)^2}{2\sigma^2})$ 由两个参数控制：均值 $\mu$ 决定峰值位置，方差 $\sigma^2$ 决定散布宽度。高斯分布在 AI 中无处不在的最深层原因是中心极限定理——大量独立随机因素的叠加天然趋向高斯分布，而真实世界数据中的"噪声"往往就是许多微小不确定因素的混合。均方误差损失函数本质上就是假定误差服从高斯分布下的负对数似然。

多元高斯分布 $\mathcal{N}(\boldsymbol{\mu}, \boldsymbol{\Sigma})$ 将高斯推广到高维：$p(\mathbf{x}) = \frac{1}{(2\pi)^{d/2}|\boldsymbol{\Sigma}|^{1/2}} \exp(-\frac{1}{2}(\mathbf{x}-\boldsymbol{\mu})^\top\boldsymbol{\Sigma}^{-1}(\mathbf{x}-\boldsymbol{\mu}))$。协方差矩阵 $\boldsymbol{\Sigma}$ 不仅描述每个维度自身的方差，还编码了维度之间的线性关联——$\Sigma_{ij} > 0$ 意味着当第 $i$ 维高于均值时，第 $j$ 维也倾向于高于均值。

Beta 分布 Beta$(\alpha,\beta)$ 定义在概率区间 $[0,1]$ 上，形状由两个参数控制，既可以是钟形也可以是 U 形。它在 AI 中的重要性源于它是二项分布和伯努利分布的共轭先验——这意味着如果先验是 Beta，观测到二项数据后后验仍然是 Beta，更新规则就是简单的参数加法。Dirichlet 分布是 Beta 的多元推广，在主题模型如 LDA 中用于主题-词分布的先验。

# 期望、方差与矩

期望和方差是描述随机分布的两个最重要的数字特征。期望 $\mathbb{E}[X]$ 是分布在概率加权下的"重心"，也就是长期平均意义上的取值；方差 $\text{Var}(X) = \mathbb{E}[(X - \mathbb{E}[X])^2]$ 衡量分布围绕均值的散布程度。

协方差 $\text{Cov}(X,Y) = \mathbb{E}[(X - \mathbb{E}[X])(Y - \mathbb{E}[Y])]$ 量化两个变量共变的方向和强度——正值意味着同步增减，负值意味着反向变化。归一化后的相关系数 $\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sqrt{\text{Var}(X)\text{Var}(Y)}}$ 取值在 $[-1,1]$ 之间，但需要注意它只捕捉线性关系——$X$ 和 $X^2$ 有确定的函数关系但相关系数为零。

条件期望 $\mathbb{E}[Y|X=x]$ 给出"已知 $X$ 取某个值后 $Y$ 的平均预测"，这在统计学上是均方误差意义下的最佳预测。迭代期望定律（也称 Adam 定律）$\mathbb{E}[Y] = \mathbb{E}[\mathbb{E}[Y|X]]$ 揭示了条件期望与无条件期望之间的解耦关系——这个恒等式在 EM 算法的推导和强化学习的贝尔曼方程中起关键作用。

随机变量函数的期望无需重新推导新分布——使用 LOTUS 规则直接从原分布计算：$\mathbb{E}[g(X)] = \int g(x) p_X(x) dx$（连续），或 $\sum g(x) P(X=x)$（离散）。

# 指数族与共轭先验

指数族是 AI 中最重要的一族概率分布，它将大量常见分布——高斯、伯努利、多项、泊松、Gamma、Beta——统一在一个数学形式下：
$$p(x|\boldsymbol{\eta}) = h(x) \exp\{\boldsymbol{\eta}^\top\mathbf{T}(x) - A(\boldsymbol{\eta})\}$$

其中 $\boldsymbol{\eta}$ 是自然参数，$\mathbf{T}(x)$ 是充分统计量（捕获数据中关于参数的全部信息），$A(\boldsymbol{\eta})$ 是对数配分函数（保证归一化）。这个统一形式带来了几个关键推论：

指数族分布的对数似然 $\ell(\boldsymbol{\eta}) = \boldsymbol{\eta}^\top\sum_i \mathbf{T}(x_i) - nA(\boldsymbol{\eta})$ 是关于 $\boldsymbol{\eta}$ 的凹函数，最大似然估计是凸优化问题，具有唯一的全局最优解。充分统计量的维度与数据量无关——无论训练集多大，只需维护充分统计量的聚合值。这些性质使得指数族模型在理论上优雅、在实践中高效。

共轭先验解决了贝叶斯更新中的计算瓶颈。当先验分布属于某个分布族，且观测数据后后验仍属于同一族时，贝叶斯更新的计算从复杂的积分退化为简单的参数更新。经典共轭对包括：Beta-Binomial（Beta 先验 + 二项似然，后验仍是 Beta）、Dirichlet-Multinomial（多项式数据的贝叶斯分析）、Gaussian-Gaussian（均值未知时的高斯推断）。共轭性在精确贝叶斯推断中极为重要——无需数值积分或采样即可得到闭式后验。
