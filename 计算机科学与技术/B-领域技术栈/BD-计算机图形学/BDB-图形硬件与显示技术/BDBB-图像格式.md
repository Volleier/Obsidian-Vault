# 介绍
图像格式是计算机存储图片的格式

# 分类
| 格式   | 全称                               | 特点            |
| ---- | -------------------------------- | ------------- |
| JPEG | Joint Photographic Experts Group | 有损压缩，适合照片     |
| PNG  | Portable Network Graphics        | 无损，支持透明，适合网页  |
| GIF  | Graphics Interchange Format      | 最多256色，支持动画   |
| TIFF | Tagged Image File Format         | 无损，高质量，适合专业印刷 |
