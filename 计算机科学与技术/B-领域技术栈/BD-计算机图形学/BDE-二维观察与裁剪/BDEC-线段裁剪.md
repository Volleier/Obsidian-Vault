# 介绍
线段裁剪目标是计算出一条给定两个端点的线段在裁剪窗口中内部的部分。


# 线段裁剪算法
## Cohen-Sutherland 算法
- 区域编码：使用4位二进制码 `(T B R L)` 表示点相对于窗口的9个区域。
    - 第1位 (L - Left)：点在窗口左侧 → `x < x_min`
        - 第2位 (R - Right)：点在窗口右侧 → `x > x_max`
    - 第3位 (B - Bottom)：点在窗口下侧 → `y < y_min`
    - 第4位 (T - Top)：点在窗口上侧 → `y > y_max`
- 算法流程：
    1. 编码：计算线段两端点 `P₀` 和 `P₁` 的区域编码 `code0` 和 `code1`。
    2. 简单接受/拒绝：
        - 接受：如果 `(code0 | code1) == 0`（即两个编码的逻辑或为0），表示两点都在窗口内，线段完全可见。
        - 拒绝：如果 `(code0 & code1) != 0`（即两个编码的逻辑与非0），表示两点在窗口的同一侧外部，线段完全不可见。
    3. 细分求交：如果上述测试失败，线段可能部分可见。选择一个位于窗口外的端点（编码非0的端点），检查其编码的每一位，确定它位于哪个外部区域，然后计算线段与对应边界线的交点。
        - 例如，如果 `code` 的 `L` 位为1，则与左边界 `x = x_min` 求交。
        - 用交点替换这个外部端点，并更新其编码。
    4. 重复：返回步骤2，用新的端点对继续进行测试。这个过程会不断剪掉位于外部的部分，直到线段被完全接受或拒绝。
