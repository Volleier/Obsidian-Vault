# 介绍

纹理（Texture）是材料与表面外观的重要属性，反映局部图像中重复、统计或结构化的模式。纹理描述了表面粗糙度、方向性、重复周期、随机性等视觉特性，常用于识别材料（布料、木材、石材等）、分割不同外观区域、检索相似图像与合成样式。

纹理表示的目标是以紧凑且判别性的方式总结局部结构的重复模式：局部统计量（能量、共生矩阵）、滤波器响应分布或局部描述子（例如 LBP、SIFT on patches）。

# 纹理分析的主要思路

- 空间/尺度上分析局部重复模式并提取稳定响应。常见流程：构建滤波器组 → 对图像卷积得到响应图 → 对响应做能量或统计汇总 → 得到特征向量用于分类/检索/分割。
- 关键问题：尺度选择（多尺度）、方向性（多方向）、对光照/仿射/噪声的鲁棒性、以及特征的维度与降维。

# 滤波器组（Filter Banks）

滤波器组是提取多尺度、多方向结构特征的有效工具。通过一组带有不同尺度与方向参数的线性滤波器对图像做响应，汇总这些响应即可得到对纹理的描述。
滤波器组通常包含：低通/高通、定向带通（Gabor）、二阶导数（Laplacian/LoG）、Gabor 的离散近似、Laws 滤波器、Leung–Malik (LM) 滤波器组、小波基以及 steerable 滤波器等。

## 常见滤波器组简介

- Gabor 滤波器：带有尺度与方向参数的正弦调制高斯包络，良好地模拟生物视觉早级响应，适合提取方向性纹理能量。
- Leung–Malik (LM) 滤波器组：一组差分高斯与一阶/二阶导数滤波器，配合不同尺度与方向，常用于经典的纹理识别基线。
- Laws 滤波器：一组小型核（例如 L5, E5, S5 的组合）用于提取局部能量响应，计算简单且效果在一定场景下稳定。
- 小波与多尺度金字塔：使用正交/冗余小波或高斯金字塔分析不同尺度的纹理分量，适合多尺度统计描述。
- Steerable 滤波器：可在任意方向上连续插值滤波响应，便于旋转不变或方向估计。

## 从滤波器响应到纹理特征

- 响应汇总：对每个滤波器响应计算能量（平方和）、绝对值均值、方差、直方图或更复杂的统计量（如高阶矩、联合直方图）。
- 池化与降维：对局部窗口做池化（平均/最大/能量池化），再用 PCA、k-means / codebook（如 textons）、Fisher Vector 或 BoVW 压缩描述子。
- 旋转/尺度不变处理：可在特征级做归一化（对响应进行方向对齐或循环不变统计），或使用 steerable / 仿射协变区域先验实现不变性。
