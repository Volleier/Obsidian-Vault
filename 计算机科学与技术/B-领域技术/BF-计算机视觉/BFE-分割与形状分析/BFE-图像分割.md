
# 介绍
-把图像划分为若干具有语义或外观一致性的区域（像素级标签、超像素或实例掩码），用于后续识别、测量、编辑或可视化任务。
- 心理学基础：人类视觉系统倾向于将场景按连贯性、相似性、连续性与闭合性分组（格式塔原则）。分割算法常借用这些先验：颜色/亮度相似性、纹理一致性、边界连续性与区域紧凑性。

## 基本范式

- 分割方法大致可分为：基于像素/聚类的方法、基于图论的方法、基于能量最小化的方法、基于模型的方法（水平集、活动轮廓）、以及基于学习的方法（深度分割网络）。

# 基于聚类的方法

### K-means

- 思路：在像素特征空间（例如 RGB、Lab、位置、滤波器响应等）上做 k-means 聚类，把每个像素分配到最近簇。优点是简单、易实现；缺点是需预设 k、对初值敏感、无法直接保证空间连通性。
- 工程提示：常把颜色与位置一起作为特征（xy + Lab）或先做超像素（SLIC）后对超像素聚类以提高效率和连通性。

### 均值漂移（Mean Shift）

- 思路：在特征-空间里用非参核密度估计寻找模态（峰值），把像素映射到峰值所在簇，自动决定簇的数量。对复杂分布更稳健，能够发现任意形状的簇，但计算开销较大。
- 应用：常用于基于颜色/空间的平滑分割与模式发现；参数为核带宽（决定尺度与合并粒度）。

# 基于图论的方法

### 最小割 / Graph Cut

- 思路：构造带权无向图，节点为像素或超像素，边权表示相邻像素之间的不相似度；通过最小割（s-t cut）在给定前景/背景种子下求最优分割，等价于图上能量最小化。常用于交互式分割与能量式分割。
- 优点：能结合数据项（像素与模型契合度）与平滑项（鼓励边界连续），有严格能量最优化基础。缺点：依赖能量设计，二值化场景下效果好，多标签需扩展（α-扩展等）。

### 归一化割（Normalized Cut，N-cut）

- 思路：把分割视为图的全局划分问题，通过最小化类内连通度/类间割断比来获得平衡的分割，常用谱方法求解近似特征向量并做聚类。
- 优点：能捕获全局结构，适合图像分割与图像分组；缺点：谱分解开销大，对噪声和参数敏感。

## 纹理特征在分割中的应用

- 纹理描述：使用滤波器组（Gabor、LM、Laws、小波等）或局部统计（LBP、响应能量直方图）提取纹理特征，作为像素/超像素的附加特征，与颜色一起输入聚类/图割/能量模型。
- 场景：纹理对颜色不明显或光照复杂的区域尤为重要（例如草地、织物、砖石）。在图割的数据项中把纹理相似性建模能显著提升分割质量。

## 不同方法的优缺点与适用场景

- K-means / 均值漂移：优点是实现简单、参数直观；适合无监督的粗分割与初始聚类。缺点是对噪声敏感、缺乏空间连通保证（可结合位置或超像素）。
- 图割 / 能量最小化：优点是能整合多种先验（数据项与平滑项），适合交互式分割与精细边界；缺点是需要仔细设计能量与参数，多标签或全局最优扩展复杂。
- 归一化割 / 谱方法：适合捕获全局结构与非局部相似性，但计算开销高，尺度受限于矩阵规模。
- 基于纹理的方法：在纹理显著而颜色不明显时优势明显，但对尺度选择、滤波器组设计与光照敏感性需注意。
- 深度学习方法（端到端分割网络，如 U-Net、DeepLab）：优点是端到端学习表示，高精度且可处理复杂上下文；缺点是需要大量标注数据与计算资源，且难以解释内部表征。
